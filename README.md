alecharris.info

This repo is a simple static site intended for GitHub Pages.

## Files

- `index.html`: page content / layout
- `styles.css`: site styling
- `assets/portrait.jpg`: your portrait photo (add/replace this file)

## Preview locally

Any static server works. For example:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
